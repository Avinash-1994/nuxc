## 🚀 How to Use These Templates

### 1. Choose Your Template

```bash
# Premium Dashboard
npx create-nuxco my-app --template premium-dashboard

# Minimalist Portfolio
npx create-nuxco my-app --template minimalist-portfolio

# E-Commerce
npx create-nuxco my-app --template ecommerce-modern

# SaaS Landing
npx create-nuxco my-app --template saas-landing

# Crypto Dashboard
npx create-nuxco my-app --template crypto-dashboard

# Magazine Blog
npx create-nuxco my-app --template blog-magazine
```

### 2. Customize

All templates are fully customizable:
- Change colors in CSS variables
- Swap out components
- Add your own features
- Modify layouts

### 3. Deploy

```bash
npm run build
# Upload dist/ to your hosting
```

---

## 💡 Customization Tips

### Change Color Scheme

```css
/* In your CSS file */
:root {
  --primary: #your-color;
  --secondary: #your-color;
  --accent: #your-color;
}
```

### Add Your Branding

```tsx
// Replace logo and text
<h1>Your Brand Name</h1>
<img src="/your-logo.svg" alt="Logo" />
```

### Modify Animations

```tsx
// Adjust Framer Motion settings
<motion.div
  initial={{ opacity: 0 }}
  animate={{ opacity: 1 }}
  transition={{ duration: 0.5 }} // Change duration
/>
```
