## 🚀 Getting Started

### Installation
```bash
npm install -g zeptr
```

### Create New Project
```bash
npx create-zeptr my-app --template premium-dashboard
cd my-app
npm install
```

### Start Development
```bash
zeptr dev
```

### Build for Production
```bash
zeptr build
```
