# Framework Adapters

Sparx supports 16 frameworks via its adapter system.

## Supported Adapters

- Angular (with Ivy compiler)
- Nuxt.js (with Nitro bridge)
- SvelteKit (with +server.ts)
- SolidStart (with renderToStream)
- Qwik City (with optimizer)
- Astro (with Islands + Content Collections)
